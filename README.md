# Seeman Auto & Agro Industries — Business Management System

A web-based business management system developed as a final year project for BSc (Hons) Software Engineering at the University of Portsmouth.

## Overview

This system is being built for Seeman Auto & Agro Industries, a Nigerian automotive parts distributor operating across multiple branches. It replaces manual record-keeping processes with a centralised digital solution featuring offline-first capabilities.

## Tech Stack

- **Frontend:** React, TypeScript
- **Backend:** NestJS
- **Database:** PostgreSQL

## Key Features (Planned)

- Authentication & role-based access control
- Inventory management across branches
- Sales tracking & reporting
- Offline synchronisation
